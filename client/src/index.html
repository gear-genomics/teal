Application Description

Teal is a tool to view sanger trace files and extract the sequence as text. Just select a file and hit submit.

Accepted Input

The trace files can be provided in abi or scf trace format (*.scf, *.abi, *.ab1, i *.ab! and *.ab).

Sample Data

The "Show Example" button loads an sample trace file (click to download file).
